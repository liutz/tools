Meteor开发环境

一、快速开始
    
    安装Meteor
    # curl https://install.meteor.com | /bin/s`

    创建xxx项目
    # meteor create xxx

    运行xxx它
    # cd xxx
    # meteor

    部署到公共网络（免费）
    # meteor deploy try-meteor.meteor.com`

    注：参考https://github.com/meteor/meteor/

二、安装编辑器
    https://github.com/atom/atom
    安装完成后，在当前meteor项目下执行
    # atom .

三、安装mongo,查看后台数据（数据库管理）
    http://www.robomongo.org/
    meteor项目启动后台端口号如3000，mongo端口一般是3001

四、安装chrome浏览器（调试工具）
    按F12打开chrome控制台
